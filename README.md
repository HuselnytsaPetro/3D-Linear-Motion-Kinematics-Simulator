# Симулятор кінематики прямолінійного руху у 3D

Інтерактивний веб-застосунок для моделювання та візуалізації прямолінійного рівноприскореного руху матеріальної точки у тривимірному просторі засобами Three.js.

---

## Опис

Застосунок дозволяє задати початкові фізичні параметри руху та спостерігати траєкторію і кінематичні характеристики в реальному часі у 3D-сцені з можливістю інтерактивного обертання камери.

Рух описується двома кутами:
- **θ (азимут)** — напрямок у горизонтальній площині XZ
- **φ (кут підйому)** — відхилення від горизонталі вгору/вниз

---

## Вхідні параметри

| Параметр   | Опис                   | Діапазон        |
|------------|------------------------|-----------------|
| x₀, y₀, z₀ | Початкові координати   | [-1000; 1000] м |
| θ          | Кут азимуту            | [0°; 360°]      |
| φ          | Кут підйому            | [-90°; 90°]     |
| v₀         | Початкова швидкість    | [0; 500] м/с    |
| a          | Прискорення            | [-50; 50] м/с²  |
| T          | Тривалість моделювання | [0.1; 60] с     |

---

## Математична модель

Вектор напряму:
```
dx = cos(φ) · cos(θ)
dy = sin(φ)
dz = cos(φ) · sin(θ)
```

Рівняння руху:
```
x(t) = x₀ + v₀·dx·t + ½·a·dx·t²
y(t) = y₀ + v₀·dy·t + ½·a·dy·t²
z(t) = z₀ + v₀·dz·t + ½·a·dz·t²
v(t) = v₀ + a·t
```

## Структура проекту

```
linear_motion_3d/
├── index.html        # розмітка, підключення бібліотек і модулів
├── style.css         # стилі, темна тема, overlay-панелі
└── js/
    ├── physics.js    # обчислення вектора напряму і траєкторії
    ├── renderer.js   # Three.js сцена, лінія, маркер, анімація
    ├── ui.js         # форма, валідація, інфо-панель
    └── main.js       # точка входу, координація модулів
```
